‚ùì Troubleshooting

Chrome Won't Launch?

  • Check Chrome is installed
  • Try manual browser config with proxy settings
  • Use Firefox or Edge instead

No Requests in History?

  • Verify proxy is running (green status)
  • Check browser proxy settings
  • Try visiting HTTP site first (not HTTPS)

Passive Scan Not Finding Anything?

  • Enable "Passive Scanning" checkbox
  • Visit pages with forms, cookies, data
  • Some issues only appear on specific page types
